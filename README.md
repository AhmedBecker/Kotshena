# Kotshena
Data Structure Project of EECE 2024 that contains some local egyptian card games in addition to foreign games like:
1. **البصرة**
2. **فلوس**
3. **BlackJack/21**

# Algorithm and description: 
cards: 
0 = NULL            

1 to 10 = numbers 

11 = Jack (الولد) 

12 = Queen (البنت)

13 = King (الشايب)
